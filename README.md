This project is a Spotify Web Player Clone built using only HTML and CSS. It replicates the basic layout and user interface of Spotify, focusing on frontend design and styling skills.

The application includes a sidebar navigation, music library section, trending and featured playlists, and a music player UI at the bottom. It is designed to closely resemble the original Spotify interface while maintaining a simple and clean structure.

The project demonstrates strong understanding of CSS Flexbox, layout structuring, responsive design, and UI styling. It also uses modern design elements such as cards, sticky navigation, and custom progress bars to enhance the user experience.
